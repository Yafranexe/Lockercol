Digital Electronic Lock

This lock can be programmed to work in public or private mode and allows code combinations between 4 and 16 digits.

Public mode: a code must be assigned for each opening and closing. Private mode: a single key must be assigned for opening. It is important to remember that this lock can be configured to operate either in PRIVATE mode or in PUBLIC mode, that is, it allows switching between both modes. Material: Zinc Alloy Size: 38*38*10mm Weight: 35g Bolt length between lock and panel: 25mm 15mm 20mm Battery: DC 3V (button-included) Unlock mode: digital code Response time: less than 20ms External emergency power supply: Micro-USB connector; 3 ~ 4.5V (not included) Once the purchase is made, installation manual (very simple) and user manual will be sent. Lock body - Retractable type - Material: Zinc alloy and Cold rolled steel - Surface treatment: matte chrome
